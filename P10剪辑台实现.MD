**D-2 — VideoEditorStage.tsx：导出状态机**

在 `VideoEditorStage` 内新增本地状态：

```ts
const [renderState, setRenderState] = useState<{
    status: 'idle' | 'pending' | 'rendering' | 'completed' | 'failed'
    progress: number
    error: string | null
    outputUrl: string | null
}>({ status: 'idle', progress: 0, error: null, outputUrl: null })

const pollTimerRef = useRef<ReturnType<typeof setInterval> | null>(null)
```

`handleExport` 替换为：

```ts
const handleExport = async () => {
    setRenderState({ status: 'pending', progress: 0, error: null, outputUrl: null })
    try {
        await startRender(project.id)
        // 开始轮询
        pollTimerRef.current = setInterval(async () => {
            const rs = await getRenderStatus(project.id)
            setRenderState({
                status: rs.status,
                progress: rs.progress ?? 0,
                error: rs.error ?? null,
                outputUrl: rs.outputUrl ?? null,
            })
            if (rs.status === 'completed' || rs.status === 'failed') {
                clearInterval(pollTimerRef.current!)
            }
        }, 2000)
    } catch {
        setRenderState({ status: 'failed', progress: 0, error: t('editor.export.startFailed'), outputUrl: null })
    }
}
// useEffect cleanup
useEffect(() => () => { if (pollTimerRef.current) clearInterval(pollTimerRef.current) }, [])
```

工具栏导出按钮区域替换为状态感知 UI：

```
idle/saved        → [导出视频] 按钮
pending/rendering → 进度条 + 百分比文字 + 不可点
completed         → ✅ [下载] 链接 + [重新导出] 按钮
failed            → ❌ 错误摘要 + [重试] 按钮（重新调用 handleExport）
```

## 6. 阶段10目标流程（建议版）

### 6.1 E2E 主流程

1. 用户进入阶段10页面（projectId + episodeId）。
2. 系统加载 editor project：
   - 若存在则加载并迁移到最新 schema。
   - 若不存在则根据阶段8/9结果自动创建初始时间轴。
3. 用户完成编辑：片段顺序、转场、BGM、字幕样式、调色参数。
4. 用户保存项目（PUT /editor）。
5. 用户发起导出（POST /editor/render）。
6. 前端轮询导出状态（GET /editor/render?id=...）。
7. 成功后返回 outputUrl，支持下载/发布。
8. 失败时给出错误原因与重试入口。

### 6.2 导出状态机

pending -> rendering -> completed
pending -> rendering -> failed
failed -> rendering（重试）

约束：

1. 同一 editorProjectId 同时只能有一个活跃渲染任务。
2. 重试应保留最近一次失败信息与日志索引。
