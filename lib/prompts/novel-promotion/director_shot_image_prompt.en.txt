[FOUR-FRAME TIME PROGRESSION]

- If the shot is static: all 4 frames nearly identical, only tiny expression changes
- If the shot has camera movement: LT→RB reflects the framing/angle shift from start to end
- If there is action: LT→RB reflects the action from start to completion
